# D3-Challenge
## Project Overview:
Tools/Requirements
The tools used for this project are:

D3.js, 

Bootstrap,

jquery,

popper.js

#### Environment requirements:
bootstrap === 4.1.3,

d3.js === v5.16.0,

popper.js === 1.14.3, 

jquery === 3.3.1


## Objectives
In this project I created a scatter plot between individuals who smoke and individuals age. Using D3 techniques I created a scatter plot that represents each state with circle elements. I included state abbreviations in the circles. This project was an exersise in using D3 to creat custom visualizations. 

#### Data sourcing and cleaning
The data was written to a csv and read into javascript to clean and create the D3 graph.
